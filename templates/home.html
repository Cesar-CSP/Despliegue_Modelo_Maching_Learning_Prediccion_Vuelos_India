Documentación Oficial de la API de Predicción de Vuelos (India)

Bienvenido a la documentación técnica de nuestra API. Este servicio utiliza un modelo de Machine Learning (XGBoost) entrenado para predecir precios de vuelos en clase económica.

Ir al Predictor Visual ✈️

Endpoints Disponibles

GET /

Función: Landing page principal (esta página). Contiene la documentación y guía de uso de los servicios.

GET /predict

Función: Interfaz gráfica de usuario (HTML). Permite realizar predicciones de forma visual mediante un formulario.

GET /api/v1/predict

Función: Endpoint técnico para obtener predicciones en formato JSON.

Parámetros requeridos:

ParámetroTipoDescripción
airlineStringAerolínea (Vistara, AirIndia, Indigo, etc.)
fromStringCiudad de origen (Delhi, Mumbai, etc.)
toStringCiudad de destino
dep_time_catStringCategoría hora salida (Morning, Evening, etc.)
arr_time_catStringCategoría hora llegada (Night, Afternoon, etc.)
stop_numIntNúmero de escalas (0, 1, 2)
days_leftIntDías de antelación para el vuelo

GET /api/v1/retrain

Función: Re-entrenamiento automático del modelo. Busca nuevos datos en data/economy.csv, ajusta los parámetros y devuelve las nuevas métricas (RMSE, MAE, R2).
